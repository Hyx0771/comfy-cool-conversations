üìã Basic Integration

Add this script tag to your website to include the Clobol chat widget:

<script 
  src="https://your-domain.com/embed.js"
  data-mode="faq"
  data-theme="light"
  data-position="bottom-right"
  data-primary-color="#007BFF"
></script>

‚öôÔ∏è Configuration Options

Data Attributes

  • data-mode: "faq", "quote", or "support" (default: "faq")
  • data-theme: "light" or "dark" (default: "light")
  • data-position: "bottom-right", "bottom-left", "top-right", "top-left" (default: "bottom-right")
  • data-primary-color: Any valid CSS color (default: "#007BFF")
  • data-title: Widget title (default: "Clobol Support")
  • data-subtitle: Widget subtitle (default: "Hoe kunnen we helpen?")

JSON Configuration

For more complex configuration, use the data-config attribute:

<script 
  src="https://your-domain.com/embed.js"
  data-config='{"mode":"quote","theme":"dark","primaryColor":"#ff6b6b"}'
></script>

üîß JavaScript API

Control the widget programmatically with JavaScript:

// Open the widget
ClobolWidgetAPI.open();

// Close the widget
ClobolWidgetAPI.close();

// Update configuration
ClobolWidgetAPI.updateConfig({
  theme: 'dark',
  primaryColor: '#ff6b6b'
});

// Destroy the widget
ClobolWidgetAPI.destroy();
