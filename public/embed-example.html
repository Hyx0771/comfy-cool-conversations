Bolt Chat Widget - Embed Instructions

🚀 Quick Start

To embed the Bolt chat widget on your website, simply add this script tag before the closing </body> tag:

<script src="https://your-domain.com/embed.js"></script>

📋 Complete Example

Here's a complete HTML example:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My Website</title> </head> <body> <h1>Welcome to My Website</h1> <p>This is my website content...</p> <!-- Bolt Chat Widget --> <script src="https://your-domain.com/embed.js"></script> </body> </html>

⚙️ Configuration

The widget is automatically configured, but you can customize it if needed:

<script> // Optional: Configure before loading window.BoltChatConfig = { baseUrl: 'https://your-domain.com', position: 'bottom-right', // bottom-left, bottom-right, top-left, top-right theme: 'light' // light, dark }; </script> <script src="https://your-domain.com/embed.js"></script>

🔧 Deployment Steps

  1. Deploy your Lovable app to a domain (e.g., https://your-app.lovable.app)
  2. The embed.js file will be available at https://your-app.lovable.app/embed.js
  3. Include the script on any website where you want the chat widget
  4. The widget will automatically appear in the bottom-right corner

📝 Note

Replace https://your-domain.com with your actual deployed domain. The widget will automatically load the chat interface from your deployed app.

🌐 Cross-Origin Considerations

The widget uses an iframe to load the chat interface. Make sure your domain allows iframe embedding if you're using additional security headers.
