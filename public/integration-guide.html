Quick Start

Add the Clobot widget to your website with just one line of code:

<script src="https://clobot-widget.netlify.app/embed.js"></script>

Basic Configuration

Customize the widget behavior by setting configuration before loading:

<script> window.ClobotConfig = { position: 'bottom-right', // 'bottom-right', 'bottom-left', 'top-right', 'top-left' theme: 'auto', // 'light', 'dark', 'auto' primaryColor: '#2563eb', // Your brand color borderRadius: '12px', // Rounded corners initialMode: 'welcome', // 'welcome', 'quote', 'support', 'faq' showBranding: true, // Show "Powered by Clobot" language: 'en' // Language code }; </script> <script src="https://clobot-widget.netlify.app/embed.js"></script>

Advanced Configuration

Custom Styling

window.ClobotConfig = { primaryColor: '#ff6b35', borderRadius: '8px', customStyles: { 'font-family': 'Your Custom Font, sans-serif', 'shadow': '0 4px 20px rgba(0,0,0,0.15)' } };

Programmatic Control

// Open the widget ClobotWidget.open(); // Open with specific mode ClobotWidget.setMode('quote'); // Close the widget ClobotWidget.close(); // Destroy the widget ClobotWidget.destroy();

Integration Examples

WordPress

// Add to your theme's footer.php or use a plugin like "Insert Headers and Footers" <script> window.ClobotConfig = { position: 'bottom-right', primaryColor: '#0073aa' // WordPress blue }; </script> <script src="https://clobot-widget.netlify.app/embed.js"></script>

Shopify

// Add to your theme's theme.liquid file before </body> <script> window.ClobotConfig = { position: 'bottom-left', initialMode: 'support' }; </script> <script src="https://clobot-widget.netlify.app/embed.js"></script>

React Application

// useEffect hook to load widget useEffect(() => { window.ClobotConfig = { position: 'bottom-right', theme: 'auto' }; const script = document.createElement('script'); script.src = 'https://clobot-widget.netlify.app/embed.js'; document.head.appendChild(script); return () => { if (window.ClobotWidget) { window.ClobotWidget.destroy(); } }; }, []);
